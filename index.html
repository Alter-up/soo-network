FAQ

How does this work?

Simply log in to SOON, post a tweet about SOON, and then join the SOON Discord channel to request ETH for free.

Are there rate limits?

Yes, you can only request once per 24 hours for each token.

What if my request keeps failing?

To prevent potential misidentification as a bot, kindly refrain from frequent operations and allow a few minutes before attempting the next action. If issues persist, feel free to seek assistance on Discord.
